使い方 / FAQ

現在の Chrome のクッキーを paprika hub の /hosts/{host} に PUT します。送ったホストはハブ側で options.cookies_from="example.com" として ジョブから参照できます。

送られるもの: name, value, domain, path, expires, secure, httpOnly, sameSite。
送られないもの: Login Data, IndexedDB, Local Storage, saved passwords — これらは Chrome 拡張 API では読めません。 完全な profile は CLI (paprika-client upload-profile) を 使ってください。

オープン中の Chrome の状態をそのまま送るので、Chrome を閉じる必要も なし、tarball を作る必要もなし。即時。
